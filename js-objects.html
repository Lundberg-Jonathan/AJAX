JavaScript objects have attributes, just like any real-world object. A person has height, weight, etc. A JavaScript object can have height, width, etc. Today we will be looking over some of the more common JavaScript objects, arrays and strings, as well as some of their attributes.

Array

Use the field below to add to the array. Arrays have an attribute called length which gives us the size of the array (how many items are in the array). Each time you add something to the array using this field, the length of the array increases.
